# node新手课（18）首页列表

# 简介

上节课我们讲了点赞，今天我们来讲查询首页列表。

# 需求

用户打卡小程序的首页就以列表的形式展示所有用户打卡发布的图片和内容。用户可以点赞某一条打卡。

# 功能流程

查询数据库中card表的所有记录，后期可以扩展加入分页或分类、标签等功能。
